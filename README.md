# きょうの小さなしあわせ

今日あった「良かったこと」「癒されたこと」「嬉しかったこと」を
記録し、あとから振り返るためのWebアプリです。

## 制作目的

日常の小さな良い出来事は、時間が経つと忘れてしまいがちです。
このアプリでは、良い出来事とそのときの気持ちを記録し、
自分が何に喜びや癒しを感じるのか振り返れるようにしました。

## アプリ画面

![きょうの小さなしあわせのアプリ画面](小さな幸せサイト画面.PNG)


## 主な機能

- 良い出来事の新規登録
- 登録内容の一覧表示
- パスワードによる編集・削除
- カテゴリとしあわせ度の登録
- 改行を含む文章の投稿・表示
- 入力文字数のリアルタイム表示
- 削除前の確認メッセージ

## 使用技術

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## 工夫した点

- 改行を含む文章を登録・表示できるようにした
- パスワードをハッシュ化して保存した
- 共通処理を関数や別ファイルに分けた
- 日記を続けたくなるような、柔らかい色合いにした
- JavaScriptで文字数表示と削除確認を実装した

## ファイル構成

- `mission_6-2.php`：メイン画面とCRUD処理
- `functions.php`：共通関数
- `db_connect.php`：DB接続設定のサンプル
- `m6_table_create.php`：テーブル作成処理
- `style.css`：画面デザイン
- `script.js`：文字数表示と削除確認

## セットアップ

1. `db_connect.php`に利用環境のDB接続情報を設定する
2. `m6_table_create.php`を実行する
3. `mission_6-2.php`を開く

DB接続情報は、公開リポジトリではダミー文字列に置き換えています。
